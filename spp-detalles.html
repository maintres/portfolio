Estructura General del Proyecto

El proyecto SPP es una aplicación web PHP que gestiona información de personas privadas de libertad (PPL). La estructura del proyecto está organizada de manera modular y eficiente.

Módulos Principales

1. Gestión de Usuarios y Autenticación

  • Sistema de roles (administrador y otros roles)
  • Gestión de sesiones
  • Configuración de usuarios

2. Gestión de PPL

  • Registro y edición de datos personales
  • Gestión de prontuarios
  • Sistema de marcas corporales
  • Registro fotográfico y huellas dactilares

3. Informes y Evaluaciones

  • Informes psiquiátricos y psicológicos
  • Evaluaciones laborales y espirituales
  • Informes sanitarios
  • Observaciones y seguimiento

Características Técnicas

Frontend

Bootstrap jQuery JavaScript

Backend

PHP MySQL MariaDB

Seguridad

  • Validación de entradas
  • Protección contra SQL Injection
  • Control de acceso basado en roles
  • Registro de auditoría de acciones
